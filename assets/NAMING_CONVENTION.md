# Convención de Nombres para Assets

Para garantizar la compatibilidad con servidores, mejorar el SEO y mantener la organización del proyecto, todos los archivos dentro de esta carpeta deben seguir las siguientes reglas:

## 1. Kebab-case (Obligatorio)
Usa minúsculas y separa las palabras con guiones medios `-`.
- **Correcto:** `libro-personalizado-azul.jpg`
- **Incorrecto:** `Libro Personalizado Azul.jpg`, `libro_personalizado_azul.jpg`, `libroPersonalizadoAzul.jpg`

## 2. Sin Caracteres Especiales
No utilices espacios, tildes, eñes o símbolos especiales.
- **Correcto:** `envio-bogota.png`
- **Incorrecto:** `envió-bogotá.png`, `envio bogota.png`

## 3. Descriptivo pero Conciso
Los nombres deben describir el contenido del archivo de forma clara para ayudar al SEO.
- **Ejemplo UI:** `icon-whatsapp.svg`
- **Ejemplo Producto:** `heritage-cuero-marron-frontal.jpg`
